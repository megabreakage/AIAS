## 1. Audit Module

Create a completly featured *Tenant-Scoped* **Audit** module with the following table columns:

- `id`: primary key, auto-increaments,(not fillable, generated by the database)
- `identifier`: uuid, unique, indexed,not null (not fillable on Model, generated in the creating event: Defined in the BaseModel's creating event)
- `tenant_id`: required, string, indexed,
- `reference_number`: string, nullable, unique (generate reference number in the creating event, format: AUD-{Year}{audit_id}-{time})
- `name`: required, string, not null,
- `checklist_id`: foreignId constraint to `checklists` table, nullable (assuming an audit can be created without a checklist, but can be associated later), on delete set null,
- `task_type_id`: foreignId constraint to `task_types` table, nullable (assuming an audit can be created without a task type, but can be associated later), on delete set null,
- `scope`: enum('internal', 'external', 'service_provider', 'supplier'), default 'internal',
- `department_id`: foreignId constraint to `departments` table, nullable (assuming an audit can be created without a department, but can be associated later), on delete set null,
- `audit_start_date`: timestamp, required,
- `audit_end_date`: timestamp, nullable (can be set when the audit is completed),
- `lead_auditor_id`: unsignedBigInteger, (user id of the lead auditor, nullable),
- `quality_manager_id`: unsignedBigInteger, (user id of the quality manager overseeing the audit, nullable),
- `add_apendix`: boolean, default false, (indicates if the audit has an appendix attached),
- `description`: text, nullable,
- `is_featured`: boolean, default false,
- `created_by`: unsignedBigInteger, (user id of the creator, nullable),
- `updated_by`: unsignedBigInteger, (user id of the last updater, nullable),
- `created_at`: timestamp,
- `updated_at`: timestamp,
- `deleted_at`: timestamp, nullable (for soft deletes)
